# Homebrew Tap

Custom Homebrew formulae for larrykoo711 projects.

## Installation

```bash
brew tap larrykoo711/tap
```

## Available Formulae

### bibo

Fast, local neural text-to-speech CLI.

```bash
brew install larrykoo711/tap/bibo
```
